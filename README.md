# EduText Analyzer

AI-инструмент для классификации сложности учебных вопросов на основе их текстового содержания.

## Как работает проект

Проект использует машинное обучение для анализа учебных вопросов и автоматического определения их уровня сложности (easy - легкий, medium - средний, hard - сложный).

### Основные компоненты:
- **Модель обучения**: Использует TF-IDF векторизацию текста и логистическую регрессию для классификации вопросов.
- **Обучение**: Модель обучается на размеченных данных из файла `data/sample_data.csv`.
- **Предсказание**: Для новых вопросов без указанной сложности модель предсказывает уровень и обновляет файл данных.

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Скачайте или клонируйте репозиторий.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. (Опционально) Установите pre-commit для автоматического запуска тестов перед коммитами:
   ```bash
   pip install pre-commit
   pre-commit install
   ```

## Запуск

1. Перейдите в корневую папку проекта.
2. Запустите скрипт предсказания:
   ```bash
   python src/predict.py
   ```
   Это обучит модель на существующих данных и предскажет сложности для вопросов без меток.

3. Результаты будут сохранены в `data/sample_data.csv`, и вы увидите вывод в консоли для каждого предсказанного вопроса.

## Тестирование

Для запуска тестов выполните:
```bash
pytest tests/
```

Тесты также запускаются автоматически перед каждым коммитом благодаря pre-commit hooks (установите с `pip install pre-commit` и `pre-commit install`).

## Структура проекта

- `src/model.py` - код для обучения модели
- `src/predict.py` - скрипт для запуска предсказаний
- `data/sample_data.csv` - пример данных с вопросами
- `tests/` - тесты для проверки функциональности
